# 📚 Student Management System

A beginner-friendly **Student Management System** built with **Python**. This project allows users to manage student records by performing basic CRUD (Create, Read, Update, and Delete) operations. Student data is stored in a JSON file, so records are saved even after the program is closed.

---

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ✏️ Update Student
- 🗑️ Delete Student
- 💾 Automatic Data Saving
- ✅ Input Validation

---

## 🛠️ Technologies Used

- Python 3
- JSON
- Built-in Python Modules

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/manojchoudhary11/Student-Management-System.git
```

Move to the project folder:

```bash
cd Student-Management-System
```

Run the program:

```bash
python student_management_system.py
```

---

## 💻 How to Use

1. Run the program.
2. Choose an option from the menu.
3. Add, view, search, update, or delete student records.
4. Your data is saved automatically.
